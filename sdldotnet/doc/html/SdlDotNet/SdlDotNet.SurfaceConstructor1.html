Represents a Image that can be drawn on a Sdl.Surface.

The image can be loaded from from a file, a System.Drawing.Bitmap, or a byte array.

Supported image formats follows the development cycle of SDL_Image. Currently, supported and planned supported image formats are:

  • .BMP - Windows Bitmap
  • .PNM - Portable Anymap File Format
  • .XPM - X PixMap
  • .LBM - Tagged Image File Format
  • .PCX - Z-Soft’s PC Paintbrush file format
  • .GIF - Graphics Interchange Format
  • .JPG - Joint Photographic Experts Group (JPEG)
  • .TIF - Tagged Image File Format
  • .PNG - Portable Network Graphics
  • .TGA - Truevision (Targa) File Format

Usage:

Surface image = Surface"mybitmap.jpg")
image.Blit(screen, new Rectangle(new Point(0,0),image.Size))
