Event Data

The event handler receives an argument of type TickEventArgs containing data related to this event. The following TickEventArgs properties provide information specific to this event.

Property Description
Fps Gets the FPS as of the event call. Events.FPS is an alternative.
LastTick Gets when the last frame tick occurred.
SecondsElapsed Seconds elapsed between the last tick and the current tick
Tick Gets the current SDL tick time.
TicksElapsed Gets the difference in time between the current tick and the last tick.
Type Returns Event type
UserCode User code that uniquely defines this user event.
