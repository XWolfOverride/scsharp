Remarks

Some implementations of the particle class include ParticlePixel and ParticleSprite.

Example

This is an example of implementing a new kind of particle. It creates a ParticleBox that uses blue boxes to represent particles: 

public class ParticleBox : BaseParticle
{
    public override void Render(Surface destination)
    {
        destination.DrawBox(new Rectangle((int)this.X, (int)this.Y, 100,100), Color.Blue);
    }
}
